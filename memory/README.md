# Investigation Memory

This folder contains committed episodic records of bounded investigations.
Records preserve the question, sources, retrieval dates, method, observations,
interpretation, uncertainty, and follow-ups. They are not automatically curated
knowledge.

Prefer a date-led path such as `memory/YYYY.MM/DD/<investigation-slug>/` when an
investigation needs multiple artifacts. Keep large or restricted source payloads
in `.cache/` and record stable source references here instead.
